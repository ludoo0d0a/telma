# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - Initial Release

### Added
- Initial release of the SNCF API Explorer
- Real-time train information and journey planning
- Station explorer with city-based browsing
- Favorites management for train stations
- Interactive maps with MapLibre GL integration
- Coverage area visualization
- Isochrone visualization for travel time zones
- Commercial modes exploration
- Lines information and details
- Schedules view for train stations
- Reports and statistics
- Places search functionality
- Interactive API documentation with Swagger UI
- PWA support with service workers
- Responsive design with Bulma CSS framework
- Analytics integration with React GA4
- Testing setup with Vitest and React Testing Library

### Technical
- React 19.2.3 with TypeScript
- React Router DOM 7.11.0 for routing
- Vite 7.3.0 as build tool
- Axios for HTTP requests
- SCSS/Sass for styling
- MapLibre GL and React Map GL for maps
- Swagger UI React for API documentation
